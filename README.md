# About
This project is to study the effects of the electrician shortage (if there is) on housing prices and units using data from the BLS QCEW and the Census 5-year ACS. 

Thus, we develop several things. 

As one may expect, the BLS changed their methodology after COVID. This means there is a whole separate methodology for the recent data, and there are __some__ files for the other
